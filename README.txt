JETPACK JOYRIDE - BY PARV JOSHI (2018101062)


How To Run ->
1) Open Folder In Terminal and run command --> python3 game.py


How To Play ->
 W-> To move upwards
 D-> To move Right
 A-> To move Left

 J-> Shield (Activated Every 20 Sec)
 K-> Shooting
 L-> Speed Boost
Objects ->

$ -> Coins (More Coins More Points)
#######-> Fire Beams
#->Dragon Ice Balls
>->Bullets


Modules Used ->
1)os
2)colorama
3)random
4)time
